# 📧 AmitXMail

AmitXMail is a modern temporary email (temp mail) web app that allows users to generate disposable email addresses instantly and receive emails without signup.

---

## 🚀 Live Demo
👉 https://tulsimeena.github.io/AmitXMail/

---

## 🔥 Features

- ⚡ Instant Temp Email Generator  
- 📥 Inbox (Coming Soon)  
- 📋 Copy Email  
- 📱 Responsive Design  
- 🌐 Easy to Use
